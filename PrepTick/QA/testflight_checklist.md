# TestFlight QA Checklist

- First launch / onboarding path
- Notifications permission path (allow/deny)
- Start timers (1 timer, 3 timers)
- Pause/resume + persistence after kill/relaunch
- Adjust +/- time while running + reschedule notifications
- Done state + clear/restart
- Presets create/edit/delete + favourites
- Repeat last set works after relaunch and after deleting a preset
- Settings toggles behaviour
- Reset app data (if present)
- Accessibility spot checks
